SMARTPAY CRM v3.5.0 — ПЛАВАЮЩИЙ ЧАТ И СКВОЗНОЕ ШИФРОВАНИЕ

ЧТО ИЗМЕНИЛОСЬ
1. Отдельная вкладка «Чат» удалена.
2. В правом нижнем углу появилась кнопка чата.
3. Общий чат видят все сотрудники и он хранится обычным текстом.
4. Новые личные сообщения защищены сквозным шифрованием:
   - текст шифруется в браузере отправителя;
   - в Google Sheets сохраняется только шифротекст;
   - расшифровать сообщение могут отправитель и получатель;
   - используются ECDH P-256, AES-KW и AES-GCM.
5. Старые личные сообщения, созданные до v3.5.0, остаются без шифрования.

УСТАНОВКА
1. Сохраните резервную копию текущих index.html и Code.gs.
2. Замените код Google Apps Script содержимым файла Code.gs.
3. Google Apps Script: Deploy → Manage deployments → Edit → New version → Deploy.
4. На GitHub/хостинге замените index.html, manifest.json и sw.js.
5. Откройте CRM и нажмите Ctrl+F5.
6. Каждый сотрудник должен один раз войти в обновленную CRM со своим паролем. После этого для него автоматически создастся ключ защищенного чата.

НОВЫЕ ЛИСТЫ GOOGLE SHEETS
- Chat — сообщения.
- ChatKeys — публичные ключи и зашифрованные приватные ключи.

ВАЖНО
- Не удаляйте строки из ChatKeys без необходимости. При удалении ключа пользователь может потерять доступ к старым личным сообщениям.
- При изменении пароля через профиль CRM ключ чата автоматически перешифровывается новым паролем.
- Если пользователь еще не входил в версию v3.5.0, отправить ему личное зашифрованное сообщение нельзя.
- Владелец Google Sheets видит метаданные (кто, кому и когда написал), но не текст новых личных сообщений.
- Общий чат не является приватным: его видят все пользователи CRM.
