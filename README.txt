SMARTPAY CRM v3.5.2 — CHAT RENDER FIX

ИСПРАВЛЕНО
- Устранена причина пустого окна чата.
- Исправлена функция формирования инициалов пользователей.
- Добавлена защита: при ошибке окно чата больше не остается пустым, а показывает понятное сообщение.
- Обновлен номер кеша Service Worker.

УСТАНОВКА
1. В GitHub в корне рабочего репозитория замените:
   - index.html
   - sw.js
   - manifest.json
2. Нажмите Commit changes и дождитесь завершения GitHub Pages deployment.
3. Code.gs менять и повторно развертывать не нужно, если уже установлен Code.gs версии v3.5.1.
4. Откройте сайт по адресу с параметром:
   /index.html?v=352
5. Нажмите Ctrl+F5 или Ctrl+Shift+R.
6. При необходимости: F12 → Application → Service Workers → Unregister,
   затем Storage → Clear site data.
