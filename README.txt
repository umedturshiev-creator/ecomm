SmartPay CRM v3.10.1 — исправление комментариев архива

ИСПРАВЛЕНО
1. В разделе «Архив» при нажатии на текст в колонке «Комментарии» теперь открывается полный текст.
2. Если комментарий отсутствует, но указана причина архивации, в окне показывается причина архивации вместо символа «—».
3. Если есть и комментарии, и причина архивации, они отображаются вместе.
4. То же поведение добавлено в мобильные карточки архивных мерчантов.
5. При переносе мерчанта в архив уведомление в Telegram-группу больше не отправляется. Действие по-прежнему сохраняется в истории CRM.

КАК ОБНОВИТЬ
1. В GitHub заменить index.html.
2. В Google Apps Script заменить Code.gs.
3. Обновить СУЩЕСТВУЮЩЕЕ развертывание Web App:
   Deploy / Развернуть → Manage deployments / Управление развертываниями → карандаш → New version / Новая версия → Deploy / Развернуть.
4. Новое развертывание создавать не нужно.
