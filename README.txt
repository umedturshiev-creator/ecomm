SMARTPAY CRM v3.5.4 — CHAT PASSWORD FRONT FIX

ИСПРАВЛЕНО
- Окно ввода пароля защищенного личного чата теперь всегда открывается поверх окна SmartPay Chat.
- Пока окно пароля открыто, элементы чата под ним не перехватывают нажатия.
- После закрытия или успешной разблокировки обычная работа чата восстанавливается.
- Обновлены версия приложения, manifest и номер кеша Service Worker.

УСТАНОВКА
1. В корне репозитория GitHub замените файлы:
   - index.html
   - manifest.json
   - sw.js
2. Нажмите Commit changes и дождитесь завершения GitHub Pages deployment.
3. Code.gs менять и повторно развертывать не нужно.
4. Откройте сайт по адресу:
   /index.html?v=354
5. Нажмите Ctrl+F5 или Ctrl+Shift+R.
6. Если старая версия осталась в кеше:
   F12 → Application → Service Workers → Unregister,
   затем Storage → Clear site data.

ПРОВЕРКА
1. Откройте SmartPay Chat.
2. Выберите пользователя в списке личных чатов.
3. Окно «Разблокировка личного чата» должно появиться впереди окна чата.
4. Поле пароля и кнопки должны быть доступны для нажатия.
