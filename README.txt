SMARTPAY CRM v3.6.3 — FUNNEL WITHOUT POTENTIAL

ИСПРАВЛЕНО
- Этап «Потенциальный» удален только из воронки подключения.
- Потенциальные мерчанты больше не участвуют в количестве, конверсии и определении точки остановки.
- Основные статусы и список потенциальных мерчантов в CRM сохранены без изменений.

ВОРОНКА ТЕПЕРЬ
В процессе → Договор → Интеграция → Тест → Активный

ДЛЯ КАЖДОГО ЭТАПА ПОКАЗЫВАЕТСЯ
- текущее количество мерчантов;
- среднее время нахождения на этапе;
- сколько перешло дальше;
- процент перехода;
- основная точка остановки процесса.

СОХРАНЕНО
- Ускоренная доставка чата.
- Звуковые уведомления и счетчики непрочитанных.
- Одна и две галочки прочтения.
- Смайлики и личные сообщения без повторного пароля.
- Все функции мерчантов, задач, партнеров и аналитики.

УСТАНОВКА
1. В GitHub замените index.html, manifest.json и sw.js.
2. Code.gs менять и повторно развертывать не нужно.
3. Дождитесь завершения GitHub Pages deployment.
4. Откройте /index.html?v=363.
5. Нажмите Ctrl+F5 или Ctrl+Shift+R.

ПРОВЕРКА
- В левом нижнем углу должно быть: v3.6.3 FUNNEL CLEAN DATA.
- На дашборде должна отображаться воронка из пяти этапов.
- Этап «Потенциальный» не должен влиять на расчеты воронки.
